# Checkpost-Arduino
Making an automatic sensor enabled model Traffic Stop/Checkpost


## BOM:
Arduino Uno(Clone Board): 250 INR
HC-04 Ultrasonic Board 45 INR
SG90 Servo: 80 INR
Jumpers and breadboard: 40INR

## Pins:
### Arduino Uno - > HC04

- 5v to VCC
- GND to GND
- D6 to Trig
- D7 to Echo

### Arduino Uno - > Servo

- 5v to Red wire
- GND to Black wire
- D9 to Orange wire
